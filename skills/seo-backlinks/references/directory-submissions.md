## Submission Best Practices

### Profile optimization

- **Business name:** consistent across all directories (NAP consistency)
- **Description:** unique 150-200 words per directory (avoid duplicate content)
- **Categories:** choose most relevant primary category
- **Images:** logo (square), cover image (16:9), product shots
- **Contact info:** complete every field (phone, email, address, website)
- **Social links:** add all relevant social profiles

### NAP consistency (critical for local SEO)

```
Name: [Exact business name]
Address: [Exact format, including suite/unit]
Phone: [Same format everywhere, e.g., (555) 123-4567]
```

### Description templates

**General business (~150 words):**
```
[Company Name] provides [primary service/product] for [target audience] in [location/industry]. Established in [year], we specialize in [key offering 1], [key offering 2], and [key offering 3].

Our [unique value proposition] helps clients [main benefit]. We've served [X clients/customers] achieving [notable result or metric].

Services include:
- [Service 1]
- [Service 2]
- [Service 3]

Contact us for [call-to-action] at [website] or [phone].
```

**SaaS / tech product (~150 words):**
```
[Product Name] is a [category] platform that helps [target users] [main outcome].

Key features:
- [Feature 1]: [Benefit]
- [Feature 2]: [Benefit]
- [Feature 3]: [Benefit]

Trusted by [X companies/users] including [notable customer if applicable]. [Product] integrates with [popular tools] and offers [pricing model].

Try free at [website] or schedule demo.
```

### Submission priority

- **Tier 1 first:** DA 80+ general + industry-specific high-authority + Google Business Profile essential
- **Tier 2 next:** DA 60-79 general directories, niche directories with DoFollow, review platforms in your industry
- **Tier 3 if time allows:** DA 40-59 directories, location-specific local directories, emerging niche platforms

### Submission workflow

1. **Prep (1-2h):** gather assets (logos, images, descriptions); standardize NAP; create 5-10 description variants; prepare credentials list.
2. **Batch submit (2-3h):** Tier 1 first; document each in tracking sheet; save confirmation emails.
3. **Verification (1 week):** check for verification emails; complete phone/email/postcard steps; update tracking sheet.
4. **Optimization (ongoing):** add photos; respond to reviews; update business hours/info; monitor for spam edits.

## Red Flags -- Avoid

- Spam directories (100+ listings per page)
- Irrelevant categories (damages relevance)
- Paid-only with no real users (link farms)
- Exact duplicate content across directories
- Directories with no organic traffic
- Sites with malware / security warnings

## Quality Signals

- Manual review process
- Active moderation
- Organic traffic (check Ahrefs/Semrush)
- Clean, professional design
- Engaged user base
- Mobile-responsive site
- SSL certificate
- Reasonable submission requirements

## Monitoring & Maintenance

| Cadence | Tasks |
|---|---|
| Monthly | Check unapproved submissions; respond to new reviews; update business info changes; add new photos/content; check for unauthorized edits |
| Quarterly | Audit all listings for accuracy; remove/disavow spam directories; analyze referral traffic; identify new directory opportunities; refresh descriptions |
| Annual | Comprehensive NAP audit; refresh all photos/media; rewrite descriptions; evaluate ROI of paid directories; clean up outdated/closed directories |
